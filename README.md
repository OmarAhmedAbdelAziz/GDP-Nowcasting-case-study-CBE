# GDP Nowcasting for Egypt

**Using Google Trends Alternative Data to Nowcast Egypt's Monthly GDP**

## Project Overview

This project builds a machine learning model to **nowcast Egypt's monthly GDP** using Google Trends search data. Nowcasting means estimating the current GDP before official figures are released — using real-time alternative data.
